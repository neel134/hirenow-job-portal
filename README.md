🚀 HireNow – Job Portal

HireNow is a full-stack job portal web application that allows users to explore and search for job opportunities through a modern and responsive interface. The application is built using React for the frontend, Spring Boot for the backend, and MySQL as the database, providing a scalable and efficient architecture.

The platform communicates through REST APIs to fetch and display job listings dynamically.

🌐 Features

Browse latest job listings

Search jobs by keywords

Dynamic job data using REST APIs

Responsive and modern UI

Fast React-based frontend

Secure backend built with Spring Boot

Database integration with MySQL

▶️ How to Run the Project
1️⃣ Clone the repository
git clone https://github.com/yourusername/hirenow.git
2️⃣ Run Backend (Spring Boot)

Open the backend folder and run:

mvn spring-boot:run

Backend will start on:

http://localhost:8080
3️⃣ Run Frontend (React)

Navigate to frontend folder and run:

npm install
npm start

Frontend will run on:

http://localhost:3000
